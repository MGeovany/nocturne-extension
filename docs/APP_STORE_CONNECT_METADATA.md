## Descripcion

Limit: 4,000 characters.

```text
Nocturne is a focused debugging companion for Safari. It brings network requests, console output, cookies, and site storage into one clean overlay so developers can understand what happened without jumping between DevTools panels.

When a request fails, Nocturne helps you see the full story: method, URL, status, timing, headers, payloads, response bodies, redirects, and nearby console messages. Filter noisy traffic, inspect JSON responses, review headers, and copy useful debugging context when you need to share an issue with a teammate or assistant.

Nocturne is built for developers, QA engineers, support teams, and anyone who debugs web apps in Safari.

Key features:

- Inspect fetch and XHR requests in a focused Safari overlay
- View status codes, timing, headers, request bodies, and response bodies
- Filter requests by errors, slow responses, domains, methods, and content type
- See console messages alongside network activity
- Inspect cookies, localStorage, sessionStorage, IndexedDB names, Cache Storage, and service workers
- Clear cookies and site storage when you choose
- Copy requests as cURL or fetch
- Copy a clean, structured debugging brief with sensitive headers masked by default

Nocturne has no accounts, no subscriptions, no analytics, and no backend. Captured debugging data stays local to your browser unless you choose to copy or export it.
```
